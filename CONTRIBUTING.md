# Contribution Guidelines

Thank you for your interest in contributing to Awesome Trade Shows!

## How to Contribute

1. Fork this repository.
2. Create a new branch: `git checkout -b add-trade-show-name`.
3. Add your changes.
4. Commit: `git commit -m "Add [Trade Show Name]"`.
5. Push: `git push origin add-trade-show-name`.
6. Open a pull request.

## Adding a Trade Show

Please ensure your submission meets the following criteria:

- The trade show is **currently active** (not discontinued).
- Provide a **working link** to the official website.
- Use the following format:

```
- [Trade Show Name](https://example.com/) - Brief description. City, Country. Frequency, Month(s).
```

- Place the entry in the **correct category** (by industry or by region).
- Keep descriptions concise (one sentence).
- Maintain **alphabetical order** within each section.

## Adding a Tool or Resource

- The tool/platform must be **relevant** to the trade show or exhibition industry.
- It should be **actively maintained** and publicly accessible.
- No affiliate links.

## General Guidelines

- Search existing entries to avoid duplicates.
- One trade show or resource per pull request is preferred.
- Check your spelling and grammar.
- Remove trailing whitespace.
- Make sure your text editor is set to remove trailing whitespace.

## Updating or Removing Entries

If a trade show has been discontinued or a link is broken, please open an issue or submit a pull request to update/remove it.
